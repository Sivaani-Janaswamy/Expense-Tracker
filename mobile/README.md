# Expense Tracker Mobile

## Run

```bash
cd mobile
npm install
npx expo start
```

## What To Check

- Auth flow: register, login, logout
- Expense flow: add, edit, delete
- Dashboard summary chart
- Empty states, retry states, and offline banner

## Backend Requirement

The backend must be running on port `5000`.
